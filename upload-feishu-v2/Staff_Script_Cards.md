# 神威药房 × 互联网医院 店员话术卡片

## 卡片使用说明

- 放在收银台或随身携带，遇到相关场景时参考
- 核心是"先解决患者问题，再引导使用互联网医院"
- 语气要自然，像聊天，不要像推销

---

## 场景一：医院缺药，处方流转到店

### 患者特征
- 拿着医院处方或收到短信
- 可能第一次听说"处方流转"
- 担心药不一样、价格贵

### 话术流程

**第一步：主动识别**
> "您好，是来取流转处方的吗？我帮您查一下。"

**第二步：核对确认**
> "这是XX医院XX医生开的方，药品是XXX，您核对一下。"

**第三步：交付+引导**
> "药已经准备好了。这个药医院暂时没有，下次如果还需要，可以直接在互联网医院开方，还是来我们店取，不用跑医院排队了。"

**第四步（可选）：教操作**
> "我加您个微信，把互联网医院入口发您，下次复诊方便。"

## 场景二：慢病患者，主动推荐线上复诊

### 患者特征
- 来买高血压/糖尿病等慢病药
- 常客，对店员有信任
- 可能抱怨去医院麻烦

### 话术流程

**第一步：发现契机**
> "阿姨/叔叔，这个药快吃完了吧？上次去医院是什么时候？"

**第二步：点出痛点**
> "每次去都要挂号、排队，挺麻烦的。我们有个更方便的办法..."

**第三步：介绍方案**
> "您可以用互联网医院，手机上找医生开方，我们给您备好，您直接来取。医保也能用。"

**第四步：消除顾虑**
> "医生都是本院的，和您去医院看一样。第一次我教您，以后就会了。"

**第五步：立即行动**
> "您现在方便吗？我帮您关注，下次就能直接用了。"

## 场景三：新顾客，建立联系引流

### 患者特征
- 第一次来店，或路过咨询
- 对互联网医院完全不了解
- 可能只是随便问问

### 话术流程

**第一步：热情接待**
> "您好，需要什么帮助？"

**第二步：了解需求**
> "您是要买药，还是咨询什么？"

**第三步：价值输出**
- 如果问药："这个药我们有。另外如果您需要长期吃，可以用互联网医院开方，更方便。"
- 如果问医院："您是想去XX医院吗？现在很多病可以直接在手机上看了，不用跑医院。"

**第四步：留下钩子**
> "您扫这个码，进群后有健康提醒，有用药问题也可以问。"

## 常见问题速查

| 患者说 | 店员回应 |
|--------|---------|
| "互联网医院安全吗？" | "这是XX医院官方平台，医生都是本院医生，病历和医院打通的。" |
| "我不会用智能手机" | "没关系，第一次我帮您操作，或者让家里年轻人帮忙设置好，以后点一下就行。" |
| "药价和医院一样吗？" | "双通道药品价格是一样的，而且您是我们会员，还有积分/优惠。" |
| "医保怎么报销？" | "目前双通道药品可以走医保，您把医保卡给我，我帮您确认。" |
| "医生能看到我的病历吗？" | "能，互联网医院和医院系统打通的，医生能看到您之前的就诊记录。" |
| "我想找某个医生" | "您说医生名字，我帮您查一下他在不在互联网医院出诊..." |

## 禁忌话术 ❌

| 不要说 | 为什么 |
|--------|--------|
| "你下载个APP" | 太麻烦，患者会拒绝 |
| "这个很简单" | 对患者来说可能很难，显得不耐烦 |
| "医院人太多，别去了" | 负面评价医院，不合适 |
| "你必须先注册" | 强制性语言，让人反感 |

## 推荐话术 ✅

| 要说 | 效果 |
|------|------|
| "我帮您..." | 体现服务 |
| "第一次我教您" | 降低门槛 |
| "下次就方便了" | 强调长期价值 |
| "您方便的时候..." | 尊重患者节奏 |

## 紧急情况处理

**处方流转失败怎么办？**
1. 安抚患者："系统有点延迟，我帮您查一下"
2. 联系互联网医院运营人员（电话：XXX）
3. 记录患者信息，稍后回访

**患者投诉怎么办？**
1. 先倾听，不要辩解
2. 表示理解："确实给您添麻烦了"
3. 立即解决或上报，承诺回复时间
4. 24小时内回访确认

| 数字 | 含义 |
|------|------|

| **0门槛** | 让患者感觉没有负担，愿意尝试 |
