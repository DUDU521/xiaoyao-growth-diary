# 互联网医院 × 神威药房 合作启动沙龙方案

## 沙龙目标

- 让神威店员理解互联网医院是什么、对他们有什么好处
- 掌握3个核心场景的引导话术
- 建立双方团队的协作默契

---

## 基础信息

| 项目 | 内容 |
|------|------|
| 时间 | 建议工作日下午 14:00-15:30（门店相对空闲） |
| 地点 | 神威医院对面门店会议室/休息区 |
| 参与人员 | 神威店员（3-5人）、互联网医院运营人员、医生代表（可选） |
| 规模 | 小型，便于互动 |

---

## 沙龙流程（90分钟）

### 第一部分：破冰+背景（15分钟）

**主持人开场：**
> "今天不是培训，是聊天。我们想和神威一起，让患者看病取药更方便，同时给门店带来新客流。"

**核心信息：**
- 互联网医院 = 手机上看病、开方、续方
- 双通道处方 = 医院开的方，可以到神威取药
- 对神威的好处：新患者来源、增加处方量、提升会员粘性

### 第二部分：场景演示（30分钟）

**现场演示3个真实场景：**

#### 场景1：医院缺药，流转到神威
**演示流程：**
1. 医生在医院系统开方 → 提示"该药可至神威药房取"
2. 患者收到短信 → 点击链接查看处方
3. 患者到店 → 店员扫码确认 → 取药

**店员话术：**
> "您好，这是从XX医院流转过来的处方，我帮您核对一下...药已经准备好了，这是您的药。下次复诊可以直接在手机上找医生开方，还是来我们店取。"

#### 场景2：慢病复诊，线上开方
**演示流程：**
1. 患者手机进入互联网医院 → 选择医生 → 视频/图文问诊
2. 医生开方 → 患者选择"神威药房取药"
3. 神威系统收到处方 → 备药 → 通知患者

**店员话术：**
> "阿姨，您这个高血压药快吃完了吧？下次不用跑医院挂号了，手机上就能找医生开，我们给您备好，您直接来取就行。我教您怎么操作..."

#### 场景3：会员引流，建立联系
**演示流程：**
1. 患者到店咨询 → 店员推荐互联网医院
2. 扫码关注 → 加入患者群/成为会员
3. 后续定期提醒复诊、用药指导

**店员话术：**
> "我们有个患者群，医生助理会在群里发健康提醒。您扫码进群，以后有用药问题可以随时问，复诊开药也方便。"

### 第三部分：实操练习（25分钟）

**分组练习：**
- 两人一组，一人扮演患者，一人扮演店员
- 练习3个场景的对话
- 互相点评，优化话术

**常见问题预演：**
| 患者疑问 | 店员回应 |
|---------|---------|
| "互联网医院靠谱吗？" | "这是XX医院官方的平台，医生都是本院医生，和来医院看一样的。" |
| "我不会用手机怎么办？" | "没关系，我帮您操作，或者您让家里年轻人帮忙，下次就会了。" |
| "为什么要来你们这取药？" | "有些药医院暂时没有，我们这儿有；而且您成为会员后有优惠，还能享受用药指导。" |
| "医保能用吗？" | "目前双通道药品可以走医保，具体我帮您查一下..." |

### 第四部分：答疑+下一步（20分钟）

**Q&A环节**

**明确分工：**
| 事项 | 互联网医院负责 | 神威负责 |
|------|---------------|---------|
| 患者咨询用药问题 | 医生/药师在线解答 | 店员记录，转达 |
| 处方流转异常 | 技术对接处理 | 门店系统检查 |
| 患者投诉/建议 | 客服跟进 | 门店安抚，反馈 |
| 推广活动 | 提供素材、培训 | 门店执行、收集反馈 |

**下一步行动：**
- 本周内：确定首批流转药品目录
- 下周：启动试点，每天复盘
- 每月：数据分析，优化流程
