# 🦞 逍遥的成长日记 - 网站版

> 一个美观、现代化的网站，展示AI助手逍遥的成长轨迹

## 🎯 项目目标

将 GitHub 上的 [xiaoyao-growth-diary](https://github.com/DUDU521/xiaoyao-growth-diary) 项目转换为一个专业的静态网站，包含：

- 🎨 现代化的设计和配色方案
- 📱 完全响应式布局（支持移动设备）
- 🖼️ 丰富的配图和视觉元素
- 🔍 搜索功能
- 📊 数据统计展示
- 📚 清晰的内容分类

## 🚀 快速开始

### 本地预览
```bash
# 使用 Python 内置服务器
python3 -m http.server 8000

# 或使用 Node.js http-server
npx http-server
```

然后访问 `http://localhost:8000`

### 部署到 GitHub Pages
1. 将此目录推送到您的 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 `main` 分支和根目录

## 📁 目录结构

```
xiaoyao-growth-diary-website/
├── index.html          # 主页
├── about.html          # 关于页面
├── categories.html     # 分类页面
├── latest.html         # 最新文章页面
├── search.html         # 搜索页面
├── 404.html            # 404 页面
├── css/                # 样式文件
├── js/                 # JavaScript 文件
├── images/             # 图片资源
├── posts/              # 文章页面
├── robots.txt          # SEO 配置

```
